# QtCalculator

A basic calculator application that performs simple arithmetic operations, developed using Qt 5.15 and QML.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Simple and intuitive interface
